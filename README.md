# Titanic_Survival_Prediction
Overview
This project implements a machine learning solution to predict passenger survival during the Titanic shipwreck. Using the classic Kaggle Titanic dataset, the notebook performs exploratory data analysis (EDA), data preprocessing, and trains a classification model to identify patterns that determined passenger outcomes.

Key Features
Exploratory Data Analysis (EDA): Visualization of survival rates across different demographics using Matplotlib and Seaborn.

Data Preprocessing: Handling missing values in key columns like Age, Cabin, and Embarked.

Feature Engineering: Transformation of categorical variables (like Sex and Embarked) and scaling of numerical features using StandardScaler to optimize model performance.

Predictive Modeling: Implementation of a Logistic Regression model using Scikit-Learn.

Evaluation: Model validation using a split-train-test approach to ensure accuracy and robustness.
